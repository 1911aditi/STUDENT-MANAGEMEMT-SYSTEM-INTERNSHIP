About the System

Student Dashboard is an all-in-one management system built to simplify academic and administrative operations.

It helps institutions manage students, guides, departments, courses, projects and detailed reporting with powerful analytics and export capabilities.

Key Features

  • Student and academic management
  • Department, course and guide management
  • Project tracking and report generation
  • Advanced charts and analytics
  • Export data in multiple formats
  • Secure role-based access

Technology Stack

Frontend HTML5, CSS3, JavaScript
Backend Node.js, Express.js
Database MS Access
Charts Chart.js
Others REST API, Local Storage
